## 졸업 작품 소개

> 작품명 : InTrip(Introduce + Trip) <br>
  개발환경 : HTML + CSS + JS <br>
  작품 소개 : 여행 계획을 짜거나 어디로 놀러가야할지 모르는 사람들을 위한 여행지 소개 페이지 <br>
  작품의 특징 : 지도 api를 사용하여 소개하는 여행지의 위치 표시 및 근처 놀거리 소개 <br>
